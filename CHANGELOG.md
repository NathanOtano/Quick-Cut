# Changelog

## 0.1.0 - 2026-05-13

Première publication publique.

- Capture d’écran Windows locale.
- Fenêtre d’annotation avec outils de dessin, couleurs, texture et actions d’édition.
- Correction du comportement du raccourci global : relance d’une capture indépendante depuis la fenêtre d’annotation, mode multi uniquement pendant la capture.
- Amélioration du rendu d’image annotée pour éviter le recadrage visuel dans l’outil d’annotation.
- Notification post-capture avec suppression disque possible tout en gardant l’image en mémoire.
- Démarrage en arrière-plan via `--tray` pour éviter les instances de lanceur séparées.
