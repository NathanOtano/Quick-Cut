# Notes de release - Quick Cut 0.1.0

Première version publique de Quick Cut.

## Inclus

- capture d’écran Windows locale ;
- annotation rapide avec outils de dessin et actions d’édition ;
- raccourci global corrigé pour relancer une capture sans masquer la fenêtre d’annotation ;
- notification post-capture ;
- lancement en arrière-plan via `--tray`.

## Installation

Téléchargez `QuickCut-win-x64-v0.1.0.zip`, décompressez l’archive, puis lancez `QuickCut.Capture.exe`.

## Intégrité

Vérifiez l’empreinte avec `SHA256SUMS.txt` joint à la release.
